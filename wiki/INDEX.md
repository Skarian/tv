# TV Automation Wiki

This is the maintained synthesis layer for the TV automation repo.

## Pages

- [[TV Control]]: device target, ADB behavior, Aerial Views wallpaper control, and command conventions.
- [[Termux Setup]]: Termux dependencies and agent-populated local discovery.
- [[Agent Runbook]]: first-stop routing table for agents before TV-facing actions.
- [[Content Preferences]]: explicitly liked media, series defaults, and recurring content preferences.
- [[App Configuration]]: app versions, app settings, and install preferences.
- [[YouTube]]: YouTube presets, search, launch behavior, and playback verification.
- [[Spotify]]: Spotify Android TV launch behavior, playlist discovery, and confirmed playlists.
- [[Stremio]]: agent-directed Stremio lookup, playback launches, and fast stream switching.
- [[VLC]]: direct URL playback in VLC for Android TV.
- [[Remote Controls]]: volume, media keys, navigation keys, and soundbar behavior.
- [[Wiki Principles]]: how this repo's LLM-maintained wiki should work.
